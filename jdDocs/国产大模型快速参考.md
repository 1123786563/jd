# 国产大模型快速参考

## 智普（Zhipu / GLM）

**Provider**: `zhipu` 或 `glm`

**常用模型**:
- `glm-4` - 通用（推荐）
- `glm-4-plus` - 高质量
- `glm-4-air` - 快速
- `glm-4-flash` - 便宜

**API**: https://open.bigmodel.cn/

---

## Kimi（Moonshot）

**Provider**: `kimi` 或 `moonshot`

**常用模型**:
- `kimi-k2.5` - 最新（推荐）
- `kimi-k2.5-free` - 免费版
- `moonshot-v1-128k` - 超长文本

**API**: https://platform.moonshot.cn/

---

## Qwen（通义千问）

**Provider**: `qwen`、`tongyi` 或 `alibaba`

**常用模型**:
- `qwen-max` - 最强（推荐）
- `qwen-plus` - 平衡
- `qwen-turbo` - 快速
- `qwen2.5-72b-instruct` - 大模型

**API**: https://bailian.console.aliyun.com/

---

## 配置示例

```json
{
  "agents": {
    "zhipu-fast": {
      "provider": "zhipu",
      "model": "glm-4-air"
    },
    "kimi-main": {
      "provider": "kimi",
      "model": "kimi-k2.5"
    },
    "qwen-pro": {
      "provider": "qwen",
      "model": "qwen-max"
    }
  }
}
```

## 安装

```bash
npm install -g @opencode/cli
opencode config set zhipu.api_key "xxx"
opencode config set kimi.api_key "xxx"
opencode config set qwen.api_key "xxx"
```
