```bash
# 1. 启动飞书客户端
```

1. **飞书应用配置**:
   - 创建企业自建应用
   - 获取 App ID、App Secret、Encrypt Key
   - 配置 Webhook URL
   - 启用 Bot 功能

2. **环境变量**:
   - `FEISHU_APP_ID` - 已配置
   - `FEISHU_APP_SECRET` - 已配置
   - `FEISHU_ENCRYPT_KEY` - 已配置
   - `FEISHU_WEBHOOK_PORT` - 默认 8080

3. **服务启动**:
   - 飞书客户端：`npm run feishu`
   - 队列处理器：`npm run queue`
