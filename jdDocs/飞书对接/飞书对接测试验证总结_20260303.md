## 🚀 使用建议

### 本地开发

```bash
# 1. 使用 ngrok 暴露本地服务
ngrok http 8080

# 2. 启动飞书客户端
npm run feishu

# 3. 启动队列处理器
npm run queue
```

### 生产环境

```bash
# 使用 pm2 管理进程
pm2 start dist/channels/feishu-client.js --name feishu-bot
pm2 start dist/queue-processor.js --name queue-processor
pm2 save
pm2 startup
```

### 监控

```bash
# 查看实时日志
tail -f .tinyclaw/logs/feishu.log

# 使用 pm2 监控
pm2 monit
```
