## 下一步操作

### 1. 实际部署

```bash
# 1. 使用真实凭证配置 .env 文件
FEISHU_APP_ID=your_real_app_id
FEISHU_APP_SECRET=your_real_app_secret
FEISHU_ENCRYPT_KEY=your_real_encrypt_key

# 2. 启动飞书客户端
npm run feishu

# 3. 在另一个终端启动队列处理器
npm run queue
```

### 2. 飞书应用配置

在飞书开放平台配置 webhook:

```
URL: http://your-server:8080/webhook
事件类型: im.message.receive_v1
验证令牌: your_encrypt_key
```

### 3. 测试完整流程

1. 在飞书客户端搜索并添加你的应用
2. 发送第一条消息（需要配对）
3. 在服务器执行配对命令: `tinyclaw pairing approve <code>`
4. 发送测试消息验证完整流程
