## 📋 详细问题列表

### 1. 模块系统规则混淆 (35 个 Story)

**问题描述**:
- Automaton 使用 ESM (NodeNext)，需要 `.js` 扩展名
- TinyClaw 使用 CommonJS，不需要 `.js` 扩展名
- 当前所有 Story 都混用或错误使用

**影响范围**:
- Epic 1a-1c: 需使用 `.js` 扩展名
- Epic 2a-2b: 不需使用 `.js` 扩展名
- Epic 3: Next.js 16 默认使用 `.js` (ESM)
- Epic 4: 取决于集成的具体项目

**修正建议**:
```typescript
// Automaton (ESM) - 本地文件导入
import { Foo } from './foo.js';  // ✅ 正确
import { Foo } from './foo';      // ❌ 错误

// TinyClaw (CommonJS) - 本地文件导入
import { Foo } from './foo';      // ✅ 正确
import { Foo } from './foo.js';   // ⚠️ 可能有问题

// 外部包导入 (所有项目)
import express from 'express';    // ✅ 正确 (不需要 .js)
```
